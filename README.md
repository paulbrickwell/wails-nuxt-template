# Wails Nuxt Template

A minimal template to get you started building desktop applications with [Wails 2](https://wails.io) and [Nuxt 4](https://nuxt.com).

## Quick Start

Create a new project using this template:

```bash
wails init -n "Your Project Name" -t https://github.com/paulbrickwell/wails-nuxt-template
```

Navigate to your project and start developing:

```bash
cd "Your Project Name"
wails dev
```

## What's Included

This template provides the essentials for building a Wails desktop app with Nuxt:

- Wails 2
- Nuxt 4

## Tailwind Version

Looking for a template with the Nuxt Tailwind and Nuxt Color Mode modules out of the box? Check out the [wails-nuxt-tailwind-template](https://github.com/paulbrickwell/wails-nuxt-tailwind-template).
